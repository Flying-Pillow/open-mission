# Airport Spec Mission

This dossier contains the current mission's working specification set.

Its purpose is to define the airport control plane and the adjacent semantic and surface contracts needed to make that architecture coherent.

This is temporary.

Once Mission can publish its own stable delivered documentation through its intended workflow, the relevant material here should be promoted into the long-term product sections and this dossier can be retired.

## Scope

- airport control plane architecture
- semantic mission model and workflow runtime boundaries
- cockpit and operator surface contract
- supporting plans, examples, and checklists
